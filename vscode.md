## Hasklig
Download [Hasklig](https://github.com/i-tu/Hasklig).

Install the fonts in OSX's Font Book 

Add the following confiugration in VS Code.
```
"[haskell]": {
  "editor.fontFamily": "Hasklig",
  "editor.fontLigatures": true
}
```
